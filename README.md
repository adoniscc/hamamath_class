# 억매방 관리센터

초등학교 1~6학년 영어 학습 관리 사이트입니다.

## 포함 기능

- 학생 관리
- 출결 관리
- 영어 학습 관리
- 상담 및 알림
- AI 학습보고서 생성
- 보고서 양식 선택
- 이미지 및 PDF 저장
- 카카오톡 공유

## 보고서 양식

- 성장 스토리: 지난달 → 이번 달 → 다음 목표
- 주간 체크: 단어·리딩·리스닝·스피킹 변화
- 클래식 리포트: 점수 변화, 학습 내용, 선생님 코멘트

## Vercel 환경변수

OpenAI API 연결 단계에서 Vercel 프로젝트 설정에 직접 입력합니다.

- OPENAI_API_KEY
- OPENAI_MODEL 선택 사항

API 키는 코드에 저장하지 않습니다.
